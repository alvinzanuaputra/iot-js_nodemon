#### Installation 


1. Initialization :
   
```bash
npm init -y
```

2. Install exprees.js

```c
npm i express
```

3. Install library socket.io
```c
npm i socket.io serialport @serialport/parser-readline
```


4. Install Nodemon js
```c
npm i nodemon
```
